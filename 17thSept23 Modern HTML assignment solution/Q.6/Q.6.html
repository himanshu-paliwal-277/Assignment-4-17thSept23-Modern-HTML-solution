Difference in HTML and HTML5

HTML HTML5
HTML provide less semantics tag. HTML5 provide more semantics tag like- header,nav,section,aside,footer and etc
HTML is not support video and audio HTML5 provide video and audio tag so use of multimedia is easy
HTML provide less tag and attributes HTML5 provide more tag and attributes
HTML not provide API HTML5 provides API's
HTML not provide Accessiblity HTML5 provide Accessiblity feature
HTML JavaScript is not allow to run in background HTML5 allowing JavaScript to run in background
